# D&D Character Creator

Веб-приложение для создания персонажей D&D 5e по правилам Player's Handbook 2024.

## Возможности

- Пошаговый wizard для создания персонажа
- 10 рас из PHB 2024 (Человек, Эльф, Дварф, Полурослик, Гном, Драконорождённый, Тифлинг, Орк, Голиаф, Аазимар)
- 12 классов с подклассами
- 16 предысторий с origin feats
- 3 метода распределения характеристик (Стандартный набор, Покупка очков, Бросок)
- Выбор снаряжения
- Экспорт в PDF
- Полностью на русском языке

## Технологии

- **React 18** + **TypeScript** + **Vite**
- **Tailwind CSS** — стилизация
- **Zustand** — управление состоянием
- **TanStack Query** — кэширование API запросов
- **jsPDF** — экспорт в PDF
- **react-i18next** — интернационализация

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен сборки
npm run preview
```

## API

Приложение использует [D&D 5e API](https://www.dnd5eapi.co/) для получения данных из SRD. Дополнительные данные из PHB 2024 (новые расы, обновлённые классы, предыстории с origin feats) добавлены локально.

## Особенности PHB 2024

- **Расы**: Новые расы (Aasimar, Goliath, Orc), обновлённые расовые черты
- **Подклассы**: Выбор подкласса на 1-3 уровне в зависимости от класса
- **Предыстории**: Origin feats — каждая предыстория даёт черту 1-го уровня
- **Характеристики**: Свободное распределение +2/+1 независимо от расы
